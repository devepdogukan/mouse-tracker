# mouse-tracker

## Install

```bash
npm install --save mouse-tracker
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'mouse-tracker'
import 'mouse-tracker/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
